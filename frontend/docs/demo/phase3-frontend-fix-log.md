# A3-1/A3-2 前端问题修复与收口说明

> 任务范围：`frontend/docs/superpowers/plans/2026-05-13-caole-phase3-frontend-closure.md` Task 3（Demo Materials And Closure Notes）要求。

## A3-1 已修问题与剩余非阻塞限制

### 已修（验收可交付）

- 导航顺序：主页到事件记录到压力分析再到沟通模拟与沟通复盘的核心闭环在入口文案和按钮链路中保持一致（首页卡片“开始记录”、记录页提交后进入分析页、分析页按钮“进入沟通演练”、模拟页按钮“生成复盘报告”、复盘页可返回首页）。
- 首页入口：顶部 Logo/品牌与侧边栏、移动端标题入口均可快速回到 `home`，并保留 `记录事件` CTA。
- 记录页空描述：`RecordView` 在 `description.trim()` 为空时阻止提交，并显示“简要描述”错误提示，避免空输入直接进入后续流程。
- 模拟复盘门禁：模拟页只有在发送成功或从缓存恢复到可用模拟结果后，才显示 `生成复盘报告` 路由入口；未生成时显示禁用按钮和引导提示。
- 动态压力环：压力分析页的圆环 `stroke-dashoffset` 会随 `pressure_score` 重新计算，文本、风险标签、来源条目与建议内容保持同步。
- 后端不可用演示兜底：`submitAnalyzeRequest`、`submitSimulationRequest`、`submitReviewRequest` 在接口不可用或返回不规范时返回 `is_demo` 演示结果，给出可见 `demo_notice`。

### 剩余非阻塞限制

- 无阻塞项；截图素材已在 `frontend/docs/demo/screenshots/` 中保留并由 `verify:phase3` 校验非空。
