使用前请了解安全边界

舍友心晴仅用于宿舍压力趋势提示和沟通练习，不进行心理疾病诊断，也不评价任何舍友的人格或心理状态。

  • 压力值只用于关系压力趋势提示，不作为医学或心理诊断依据。
  • 如果出现高压力、严重冲突、持续失眠、强烈焦虑或暴力风险，请及时联系辅导员、心理老师、家人或可信任同学。
  • Demo 阶段不采集真实身份信息，演示数据使用虚拟样例。
